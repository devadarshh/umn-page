# umn-page-width-button

This is used to toggle the width of a `umn-page` between default and expanded size. Only intended to work on `desktop` and `tablet`.

## Properties

### expandIcon

| Name | Description |
| :--- | :--- |
| Description| This is the icon of the width button when the page is contracted or shrunk or in small size. |
| Type | `string` |
| Default | "expand-outline" |

### contractLabel

| Name | Description |
| :--- | :--- |
| Description| This is the label of the width button when the page is contracted or shrunk or in small size. |
| Type | `string` |
| Default | "Expand" |

### contractIcon

| Name | Description |
| :--- | :--- |
| Description| This is the icon of the width button when the page is expanded. |
| Type | `string` |
| Default | "contract-outline" |

### contractLabel

| Name | Description |
| :--- | :--- |
| Description| This is the label of the width button when the page is expaned to the larger size. |
| Type | `string` |
| Default | "Contract" |

### showLabel

| Name | Description |
| :--- | :--- |
| Description| Whether to show label with the button. |
| Type | `boolean` |
| Default | "False" |
